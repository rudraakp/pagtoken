# Zeabur-ready Flask App (Page Token Extractor) — Detailed

This version shows more page fields: name, id, category, fan_count, perms and the page access token.

Files:
- app.py
- requirements.txt
- Procfile
- Dockerfile

Deploy to Zeabur:
1) Push to GitHub and connect Zeabur (recommended).
2) Or deploy via Dockerfile — Zeabur will detect it.

Env vars:
- FLASK_SECRET (set in Zeabur settings)
- PORT (Zeabur provides this)
